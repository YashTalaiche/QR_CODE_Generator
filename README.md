# QR_CODE_Python
QR_CODE Generator Using Python Code

QR_CODE software to create QR_CODE of any Text,Link,Number,etc

Requirements:-

1. install pyqrcode 
2. install tkinter
3. install Pillow
4. install pyzbar

Intallation
  
  pip install PyQRCode
  
  pip install tkinter 
  
  pip install Pillow==2.2.1
  
  pip install pyzbar
